# projeto-integrador-hermex
Projeto Interdisciplinar 3 e 4 do curso DSM na FATEC Itapira. Tema: Automação Comercial. Sistema para rastreamento de caixas em trânsito entre filiais, com autoria feita por monitoramento de peso e lacre. PI-3 entrega software executável e PI-4 incorpora hardware IoT  no quarto semestre.
